## 🛠️ Installation

### Clone the Repository

Begin by cloning the repository to your local machine:

```bash
git clone https://github.com/RagibShariar/E-commerce-dashboard
cd project-name
```

### Install Dependencies

```bash
npm install
# or if you use yarn
yarn install
```

## 🚀 Usage

### Development Mode

Run the development server and start building:

```bash
npm run dev
# or if you use yarn
yarn dev
```
